What I've Built

IoT Firmware C / ESP-IDF

Smart Environment Monitor

Real-time air quality, temperature, and humidity monitoring station built on ESP32. Sends data to MQTT broker, visualized via custom dashboard. Runs FreeRTOS tasks for sensor polling and WiFi management.

Key Highlights
  • FreeRTOS multi-task architecture — sensor, comms, display tasks
  • Low-power deep sleep with RTC wake on schedule
  • OTA firmware updates over MQTT
  • BME680 + CCS811 dual-sensor fusion

RTOS Firmware C / STM32

RTOS Task Scheduler Demo

Custom priority-based cooperative task scheduler built from scratch on STM32F4. Demonstrates context switching, semaphores, and inter-task communication without using an OS library.

Key Highlights
  • Bare-metal context switch using Cortex-M SysTick
  • Binary semaphore and mutex primitives
  • Task priority inheritance for deadlock prevention
GitHub →
PCB KiCad

4-Layer Motor Driver PCB

Custom 4-layer PCB for a 3-phase BLDC motor driver. Designed in KiCad with proper power plane separation, controlled impedance traces for high-frequency switching, and thermal via arrays.

Key Highlights
  • DRV8323 gate driver with SPI control interface
  • Current sensing via INA240 with 80V/V gain
  • EMI filtering and TVS protection on all inputs
GitHub → Gerbers →
IoT C++ / LoRaWAN

LoRa Mesh Sensor Network

Long-range sensor mesh using SX1276 LoRa modules across STM32L0 nodes. Custom MAC layer with TDMA scheduling for synchronized multi-node transmission without collisions.

Key Highlights
  • Custom TDMA MAC — no LoRaWAN stack dependency
  • Achieved 5km range with SF12/125kHz
  • Node power: 2.1μA sleep, 12mA active
