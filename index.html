What I've Built

IoT Firmware Embedded C / LoRa

Perimeter Intrusion Detection System

LoRa-based, three-layer centralized perimeter intrusion detection system combining UART/I2C/SPI sensors, AI/ML camera modules, a 9-axis gyro sensor, and an mm-Wave motion detection sensor.

Key Highlights
  • Wireless comms over LoRa across a three-layer sensor architecture
  • Firmware for sensor data processing and event flag generation
  • FIFO-based data transmission with ISR-controlled reception
  • Demonstrated at DRDO Bhavan Delhi, Jaisalmer Army Camp & Gandhinagar Army Camp

IoT PCB Embedded C / Quectel EC200U

Railway Water Tank Level Measurement

4G LTE IoT device using a Quectel EC200U modem with an AT command stack, transmitting tank level, GPS coordinates, and battery telemetry to a CRIS server via MQTT over TCP/IP.

Key Highlights
  • 4–20mA current loop sensor via ADC with calibrated scaling
  • Low-power sleep-wake cycle — 200mA → 40mA, an 80% reduction
  • Full lifecycle: schematic design, 4-layer PCB layout, QC, deployment
GitHub →
Firmware C / SIM800L

Payment Announcer Device

STM32-based device using a SIM800L GSM module, with firmware written in C using STM32CubeIDE to announce incoming payment SMS notifications via audio.

Key Highlights
  • Interrupt-driven UART reception with circular FIFO buffer for SMS parsing
  • Audio amplification for live payment announcements
  • Non-blocking state machine for GSM health monitoring and network recovery
GitHub →
IoT C++ / LoRaWAN

LoRa Mesh Sensor Network

Long-range sensor mesh using SX1276 LoRa modules across STM32L0 nodes. Custom TDMA MAC layer for synchronized multi-node transmission without collisions.

Key Highlights
  • Custom TDMA MAC — no LoRaWAN stack dependency
  • Achieved 5km range with SF12/125kHz
  • Node power: 2.1μA sleep, 12mA active
